Requirements
==
* [Visual Studio Code](https://code.visualstudio.com)

Installation
==
* In visual studio, go to Preferences/Extensions and select "Install from VSIX".
In the file finder window, browse for the file releases/vscode-pvs-1.0.5.vsix
